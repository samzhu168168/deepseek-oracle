# Agent Content Kit 成本详细分析

## 核心结论：开源项目本身免费，成本仅来自 API 调用

---

## 一、Agent Content Kit 是什么

**完全开源免费的项目**，代码在 GitHub 上公开，MIT 协议。它不向你收一分钱。

它的工作原理是组装现有的免费/低价 API：
- 你用你自己的 API Key
- 它帮你把流程串起来
- 所有费用是你跟 API 提供商（Anthropic、OpenAI 等）之间的结算

**类比：** Agent Content Kit 是"免费的电饭煲"，但你需要自己买米（API Key）。

## 二、每一条视频的真实成本拆解

### 用 Claude Sonnet 4.6 方案（推荐，跟你的产品同一模型）

| 步骤 | 用什么 | 一条视频消耗 | 费用 |
|------|--------|-------------|------|
| 写脚本 | Claude Sonnet 4.6 | ~600 tokens | **$0.002** |
| 配音 | edge-tts（微软免费） | ~30秒语音 | **$0** |
| 背景视频 | Pexels API（免费） | 3-5段素材 | **$0** |
| 字幕生成 | 内置 Whisper | ~30秒音频 | **$0** |
| 视频合成 | FFmpeg/Remotion | 本地计算 | **$0** |
| 发布 | 浏览器自动化 | 一次发布 | **$0** |

> **一条 30 秒视频 ≈ $0.002（0.2 美分）**

### 用 GPT-4o 方案（备选）

| 步骤 | 费用 |
|------|------|
| 写脚本 | **$0.003** |
| 其余同上 | **$0** |

> **一条 30 秒视频 ≈ $0.003**

| 方案 | 大约产出 | 够用多久（1条/天） |
|------|---------|-------------------|

## 四、隐藏成本（真正要注意的）

### 1. ElevenLabs 配音 — 最容易被忽略的大头

edge-tts 免费，但声音是微软机器音，比较"塑料"。如果你要 The Oracle 那种深沉、权威的声音：

| 配音方案 | 费用 | 声音质量 |
|----------|------|---------|
| edge-tts（免费） | $0 | ⭐⭐ 机器人感 |
| ElevenLabs 免费层 | $0（10K字符/月） | ⭐⭐⭐⭐ 约10-15条/月 |
| ElevenLabs Starter | $5/月（30K字符） | ⭐⭐⭐⭐ 约30-40条/月 |
| ElevenLabs Creator | $22/月（100K字符） | ⭐⭐⭐⭐⭐ 约100条/月 |

**建议：先用 edge-tts 跑通流程，效果好再升级到 ElevenLabs。**

### 2. 服务器 — 如果要 24 小时跑

Agent Content Kit 需要一台电脑/VPS 一直开着才能定时发布：

| 方案 | 费用 |
|------|------|
| 你自己电脑不关机 | $0（不推荐，不稳定） |
| Railway/Render 免费层 | $0（可能不够用） |
| 最便宜 VPS | $5-10/月 |

### 3. Pexels API — 永远免费

Pexels 提供免费 API，无限调用。素材库里有大量墨水、火焰、烟雾等氛围素材，刚好适合 The Oracle 风格。

## 七、我的建议

**今晚用方案A起步：**

1. `git clone` Agent Content Kit
2. 用你今晚申请的 Anthropic API Key（$10 充值）
3. 配音先用免费的 edge-tts
4. 素材用免费的 Pexels
5. 先在你电脑上手动跑，生成 3-5 条看看效果
6. 效果好再考虑部署到 VPS 上 24 小时自动跑

**$10 Anthropic 充值 = 够这个项目用几年**，因为写脚本消耗的 token 极少。

真正决定视频质量的不是 LLM 成本，是：配音质量 > 画面素材 > 字幕动效。这些大部分是免费的。
