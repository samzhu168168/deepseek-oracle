# Pixelle-Video 可直接使用的脚本 + 配置指南

> **重要：** 导演版脚本无法直接粘贴到 Pixelle-Video！Pixelle-Video 只接受纯文本文案（Mode 2）或一句话主题（Mode 1）。此文件提供两种可直接使用的格式。

---

Pixelle-Video 的 AI 会自动扩展你输入的主题成完整脚本。复制下面任意一个主题，粘贴到 Pixelle-Video 左侧「内容输入」栏，选择「AI 生成内容」模式：

```
2. You scrolled 7 hours today and made nothing. 58% of young people want to quit social media but can't. Pixelle turns your ideas into videos in seconds. One prompt, all modalities. Free and open source.

3. Adobe plus Midjourney plus ElevenLabs plus ChatGPT equals $1,583 a year just to be a creator. Pixelle replaces all of them. Open source. Zero dollars. Forever.

4. 2026 is chaos. Gen Z is in existential crisis. The news is fake, the economy is broken, and the algorithm is lying. Pixelle helps you make something real in the middle of it.

5. Every tech CEO is sorry this year. Sorry for layoffs, sorry for data leaks, sorry for the election. Pixelle has no CEO, no board, no shareholders. Just open source code you can fork right now.

6. 66 percent of young Americans want to leave the country. The Don Dash. But wherever you go, you still need income. Pixelle fits in your backpack. Borderless, subscriptionless, unstoppable.

7. You made 47 TikToks this month. 2 million views. TikTok made $6,700. You got $180. Stop feeding the algorithm. Pixelle gives you the production stack to build on your own terms.

8. Some AI video startup just raised $350 million at a $4 billion valuation to build what Pixelle already does. For free. MIT license. Here now. No waitlist.

9. You have been waiting for permission. The job offer, the client, the big break. The gatekeepers are never going to say yes. Stop asking. The Pixelle pipeline is open. Just type.

```

## 方式二：纯文本文案（固定脚本模式，效果最可控）

Pixelle-Video v2025.12.08+ 支持「固定文案内容」模式。将下面的纯文本文案粘贴到输入栏，选择分句/分行分割。

**推荐分割方式：** 按「行」分割（每行 = 一个分镜）

### 视觉 Prompt Prefix 配置

在 Pixelle-Video 视觉设置中，Prompt Prefix 填入：

```
Dark cinematic style, high contrast, moody lighting, purple and cyan color palette, minimalist composition, glitch effects, futuristic tech aesthetic, 4K quality
```

### TTS 语音配置

- Edge-TTS（免费）：选 `en-US-EricNeural`（男声，年轻直接）
- 或 ElevenLabs：选 `Adam`（更深更权威）

## Pixelle-Video 完整配置速查

打开 Pixelle-Video Web 界面（`http://localhost:8501`），按以下配置：

### 1. LLM 配置（生成文案用）

你已经在 shell 里设了环境变量，Pixelle-Video 用的是 DeepSeek 的 Anthropic 兼容接口：

| 参数 | 值 |
|------|-----|
| API Key | `sk-a62ff7c3e5c749a9ae94101d55ea8900` |
| Base URL | `https://api.deepseek.com/anthropic` |
| Model | `deepseek-v4-pro[1m]` |

> **注意：** 如果 Pixelle-Video 的 LLM 下拉菜单没有这个模型，选「自定义」，手动填入以上三个值。

### 2. 图像生成配置

| 参数 | 值 |
|------|-----|
| 方案 | RunningHub 云端（如果你没本地 GPU）或 ComfyUI 本地 |
| Prompt Prefix | `Dark cinematic style, high contrast, moody lighting, purple and cyan color palette, minimalist composition, glitch effects, futuristic tech aesthetic, 4K quality` |
| 图片尺寸 | 1080×1920（竖屏） |

### 3. 语音（TTS）

| 参数 | 值 |
|------|-----|
| TTS 引擎 | Edge-TTS（免费） |
| 语音 | `en-US-EricNeural` |

### 4. 视频模板

| 参数 | 值 |
|------|-----|
| 模板类型 | `image_vertical` 或 `static_vertical`（1080×1920） |
| BGM | 内置暗系音乐 |

### 5. 工作流验证

配置完先点各个「测试连接」按钮，确认 LLM、ComfyUI/RunningHub、TTS 都通，再点「生成视频」。
