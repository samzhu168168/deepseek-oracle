# Claude → 视频工具 万能提示词

复制以下整段发给 Claude，就能无限生成推广短视频脚本。

---

## 标准模式：生成1条视频脚本

```
你是 elemental.bond 的创意总监。这是一个面向美国市场的东方命理 SaaS 产品，功能包括紫微斗数、梅花易数、易经占卜、每日运势卡片和五行配对。

请为以下主题生成一条 TikTok/Reels/Shorts 视频脚本：

【主题】：[填写：如 "紫微斗数比星座准在哪" / "五行相克为什么让你总遇渣男" / "今天适合做什么"]
【时长】：30-45秒
【风格】：[选择：好奇悬疑 / 温暖共鸣 / 颠覆认知 / 轻松幽默]

脚本格式要求：
1. 分5个场景：Hook(0-3s) → Setup(3-8s) → Body(8-25s) → Reveal(25-30s) → CTA(30-35s)
2. 每个场景包含：📺 画面描述 / 🗣 配音旁白 / 💬 字幕文字
3. 旁白用美式口语，不要用中文术语，用现代英语解释一切
4. Hook必须制造"信息缺口"让观众想继续看
5. CTA 指向 elemental.bond
6. 最后附上：标题/Hashtags/封面文字/背景音乐风格
```

## 如何使用

1. 复制上面的"标准模式"提示词
2. 把【主题】替换成你想做的选题
3. 发给 Claude
4. 把 Claude 输出的脚本复制到你的视频自动化工具里
5. 视频工具根据 📺画面描述 匹配素材，🗣旁白 生成配音，💬字幕 叠加文字
