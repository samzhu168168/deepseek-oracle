# Elemental Bond 短视频自动化工坊

## 架构总览

```
Claude (创意大脑)
  │
  ├─→ 选题策略 → 30个选题 X 3个角度 = 90条视频素材池
  ├─→ 脚本生成 → Hook(3s) + Body(25s) + CTA(5s)
  ├─→ 视觉指导 → 场景描述 + 画面建议 + 字幕关键词
  └─→ 输出格式 → JSON / Markdown 双格式，直接复制进视频工具
```

## 工作流（4步产出1条视频）

| 步骤 | 你做什么 | Claude做什么 | 耗时 |
|------|----------|-------------|------|
| 1. 选题 | 告诉Claude今天想推哪个角度 | 生成5个选题方向供你挑 | 30秒 |
| 2. 脚本 | 确认选题 | 输出完整分镜脚本(Hook/Body/CTA) | 1分钟 |
| 3. 视觉 | 可选：指定素材风格 | 搭配场景描述+画面建议+字幕 | 30秒 |
| 4. 发布 | 复制到视频工具 | 同时生成标题/Hashtag/简介 | 完成 |

## 视频工具对接

脚本输出为标准分镜格式，可直接粘贴到：
- NanoBanana / 其他AI视频工具
- CapCut / 剪映的字幕轨道
- 任意模板化视频工具的文本输入框
